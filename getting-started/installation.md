# Installation

## Host requirements

A Linux environment with Python 3.8+ installed is required to use DipDup.

Minimum hardware requirements are 256 MB RAM, 1 CPU core, and some disk space for the database.

## Installation

### poetry (recommended)

```shell
poetry init -n
poetry add dipdup
```

### pip

```shell
python -m venv .venv
source .venv/bin/activate
pip install dipdup
```
